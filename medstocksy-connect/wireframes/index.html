Anti-patterns to reject during review

Two primary CTAs
One dominant action per screen. Demote the rest to secondary.
7th tag color
Tag palette is capped. Add metadata, not a new color.
Charts on dashboard
PRD §2.8: simple numbers/bars only. No line/pie charts in V1.
Configuration on mobile
Rule 6: campaigns, reminder rules, settings are desktop-only.
Rate-limit shown only on error
Theme §5.7: WhatsApp constraints are ambient, visible at rest.
Floating labels in forms
Predictability over novelty. Labels stay above inputs.

All wireframes on this page use only the tokens above — no off-palette color, no new component patterns. If something doesn't fit here, redesign before code (Rule 17).
